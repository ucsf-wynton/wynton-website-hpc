# Wynton Protected (decommissioned)

The Wynton HPC environment decommissioned all Protected (P3 and P4) data and accounts on 2026-01-30. Only Wynton Regular (P1-P2) accounts can use the cluster as of this date.


## UCSF options for working with P3 and P4 data

 * [CoreHPC](https://it.ucsf.edu/service/corehpc) - PHI-compliant UCSF HPC environment coming early 2026
 * [Facility for Advanced Computing (FAC)](https://it.ucsf.edu/service/facility-advanced-computing-fac) - CoreHPC works with FAC storage
 * [UCSF Data Storage for Research](https://secureresearch.ucsf.edu/ucsf-data-storage-research) - Overview of data storage options and compliance at UCSF 
 * [UCSF Research Analysis Environment (RAE)](https://it.ucsf.edu/service/rae) -  a secure data hosting and compute service for UCSF researchers and their collaborators
 * [UCSF Data Center Services](https://it.ucsf.edu/service/data-center-services) - provides storage, and virtual & physical server hosting
 * [UCSF Central IT Cloud Services](https://cloudservices.ucsf.edu/) - UCSF-compliant Secure Enterprise Clouds on AWS and Azure
